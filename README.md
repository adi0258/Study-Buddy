# Study-Buddy
https://study-buddy-m9zw.vercel.app/

A web/mobile application that connects students based on shared courses, level of understanding, and availability.

## Features

- **Smart Matching** - Students matched by shared courses, learning level, and availability
- **Profile Setup** - Each user builds a profile and selects their current courses
- **Partner Suggestions** - The app recommends the most compatible study partners
- **Responsive Design** - Works on both desktop and mobile

## Tech Stack

- **Frontend:** React, JavaScript, HTML/CSS
- **Backend:** Node.js
- **Bootstrapped with:** Create React App

## Getting Started

```bash
git clone https://github.com/adi0258/Study-Buddy.git
cd Study-Buddy
npm install
npm start
```
